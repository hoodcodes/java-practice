### The first App

1. Create a file named Hello.java
2. Add the following:

```java
class Hello {
  public static void main(String[] args) {
      System.out.println("Hello World!"); 
  }
```

3. compile the java file to create a .class file
4. run using the following command:

```sh
java Hello
```
